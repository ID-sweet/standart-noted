# standart-noted
semua catatan code standar proyek

# H Standart
text-2xl → default (mobile): ±24px
sm:text-3xl → tablet kecil: ±30px
md:text-4xl → tablet besar / laptop: ±36px
lg:text-5xl → desktop besar: ±48px

# P Standart
text-base → default (mobile) ≈ 16px
sm:text-lg → tablet ≈ 18px
md:text-xl → desktop ≈ 20px
leading-relaxed → menambah line height agar teks mudah dibaca
max-w-2xl xl:max-w-5xl mx-auto → menjaga agar teks tetap terpusat dan tidak terlalu panjang di layar besar
